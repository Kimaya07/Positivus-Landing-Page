# Positivus - Digital Marketing Agency Landing Page

A modern, responsive landing page for a digital marketing agency built as an internship UI task.

## 🔗 Live Demo
[Add your deployed link here]

## 🛠️ Tech Stack
- **React** — Component-based UI
- **Vite** — Fast build tool & dev server
- **Tailwind CSS** — Utility-first styling
- **JavaScript (JSX)** — No TypeScript

## 📁 Project Structure
```
src/
├── assets/          # Images & logos
├── components/
│   ├── Navbar.jsx
│   ├── Hero.jsx
│   ├── Brands.jsx
│   ├── Services.jsx
│   ├── Process.jsx
│   ├── Testimonials.jsx
│   └── Footer.jsx
├── App.jsx
├── main.jsx
└── index.css
```

## ✨ Features
- Fully responsive (mobile + desktop)
- Sticky navbar with hamburger menu
- Animated marquee for brand logos on mobile
- Interactive accordion for working process
- Testimonials carousel with speech bubbles
- Newsletter subscription form in footer

## 🚀 Getting Started

### Prerequisites
- Node.js v18+
- npm

### Installation
```bash
# Clone the repo
git clone https://github.com/yourusername/positivus.git

# Go into the project
cd positivus

# Install dependencies
npm install

# Start dev server
npm run dev
```

### Build for Production
```bash
npm run build
```
